# CPU-Cache-Simulator-using-LRU-algorithm
CPU Cache Simulator using least recently used algorithm, a tool for simulating and visualizing the behavior of a CPU cache using the LRU algorithm allowing users to gain insights into cache performance by tracking cache hits, and misses and presenting the data in a visualized way using the JFreeChart library
